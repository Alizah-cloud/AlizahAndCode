My Projects

Project 1: Credit

The program not only validates credit card numbers but also identifies the type of card (Visa, Mastercard, American Express) based on specific number patterns and lengths. This project helped me strengthen understanding of algorithms and conditionals in C and Python.

Project 2: Scrabble

This project simulates a Scrabble-like game where I calculate the score of a word based on its letter values. It involved working with strings, arrays, and implementing basic game rules to determine the score based on Scrabble tile values.

Project 3: Readability

This project evaluates the readability of a text using the Coleman-Liau index. The goal was to compute the text's readability score, which estimates the grade level required to understand the text. I worked with string manipulation and mathematical operations to calculate this score.

Project 4: Caesar

In this project, I implemented a Caesar cipher, which is an encryption algorithms, with the Caesar cipher shifting characters by a fixed number of places. This project helped me to work with strings, loops, and encryption basics.

Project 5: Sort

In this project, I was asked to implement one or more sorting algorithms (like Bubble Sort, Merge Sort, or Quick Sort). The goal was to arrange a set of numbers or strings in increasing or decreasing order. This helped me solidify my understanding of algorithms and data structure

Project 6: Plurality

The task was to simulate a simple election in which the candidate with the most votes wins. I needed to handle input and manage vote tallies for each candidate. This project strengthened my knowledge of arrays and loops.

Project 7: Runoff

This project involves implementing a runoff election algorithm. The methods is designed to eliminate candidates in rounds until one candidate achieves a majority, handling situations like ties and preferences. This was a complex project that required an understanding of graph theory and cycles.

Project 8: Volume

In this project, I calculated the volume of a 3D object given its dimensions. This project helped me practice using mathematical formulas and loops to compute values based on user inputs.

Project 9: Filter (Less)

This is an image filter project where I applied basic filters (like grayscale or reflection) to an image. This helped me understand how to manipulate pixels and apply algorithms to images.

Project 10: Recover

In this project, I recovered lost images from a memory card (which is presented as a series of corrupted files). The goal was to identify and extract JPEG files from a sequence of raw data. This project focuses on file handling and binary data processing.

Project 11: Inheritance

This project involves simulating inheritance in an object-oriented programming context. I created a family tree using classes to represent individuals and their relationships. The task emphasizes the understanding of object-oriented principles like inheritance, polymorphism, and encapsulation.

Project 12: Speller

The objective was to write a program that checks the spelling of words in a text file. I implementEd a spell-checker using techniques like hash tables or tries. This helped me work with string manipulation, file I/O, and memory management.

I needed to parse a DNA sequence, search for specific motifs, and compare them to a known database of sequences. This project introduced me to basic string manipulation, arrays, and algorithms for searching.

Project 14: Songs

This project involved creating a program that can store, search, and retrieve information about songs. The goal was to create a music database that organizes songs by title, artist, and year.
